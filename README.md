# Global Superstore Sales Dashboard

Excel dashboard analyzing sales, profitability, returns and shipping for a global retailer across 40+ countries.

## Objective
Help leadership identify profitable regions, understand 
return rates, and evaluate shipping strategies through 
a clear visual business story.

## What's inside
- KPIs: total sales, profit, quantity, and shipping cost
- Trends over time
- Geographic breakdown by region and country
- Category and sub-category performance
- Return rate analysis
- Interactive filters and slicers

## Tools
Microsoft Excel — pivot tables, dynamic charts, slicers

## Dataset
Global Superstore transactional data: 50,000+ orders 
across markets in Americas, Europe, Asia Pacific, and Africa.
